# TiqTaqo Frontend 🛍️

موقع TiqTaqo للتجارة الإلكترونية - الواجهة الأمامية

## 📋 نظرة عامة

موقع إلكتروني لبيع الإكسسوارات والمنتجات للرجال والنساء.

## 🚀 التقنيات المستخدمة

- HTML5
- CSS3
- JavaScript (Vanilla)
- Static Site (موقع ثابت)

## 📁 هيكل المشروع

```
tiqtaqo-frontend/
├── public/              # ملفات الموقع الرئيسية
│   ├── index.html      # الصفحة الرئيسية
│   ├── css/            # ملفات التنسيق
│   ├── js/             # ملفات JavaScript
│   └── admin/          # لوحة التحكم
├── netlify.toml        # إعدادات Netlify
└── README.md           # هذا الملف
```

## 🌐 النشر

الموقع منشور على Netlify:
- **الرابط**: https://tiqtaqo-c59d3011.netlify.app

## 🔧 التطوير المحلي

لتشغيل الموقع محلياً:

```bash
# افتح ملف index.html مباشرة في المتصفح
# أو استخدم خادم محلي
python -m http.server 8000
# ثم افتح http://localhost:8000/public
```
